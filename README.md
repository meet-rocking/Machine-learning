# Q-learning 

Create single agent reinforcement learning environment
Train agent using Q-learning

## Agent(Police)
colour --> green

Action --> 4 (move only in diagonals)

reward --> -10 at each step, +50 for catching enemy


## Enemy
colour --> red

Action --> 4 (move randomly in environment)
